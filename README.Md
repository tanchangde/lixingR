# lixingR

R 语言封装的理性仁接口。

## 动机

* 方便产出可重复、可复用、版本可控的投资研究报告
* 减少同行者重复数据搜集劳动
* 练习上规模接口封装
* 积累 R package 作品

## 目标

* 初级，实现大陆股票下个人关注的基本面指标查询，如 PB、PE、ROE、负债率、股利支付率。
* 中级，打包成 R package。
* 中高级，实现大部分实用指标查询。
* 高级，提供交互式分析服务。
